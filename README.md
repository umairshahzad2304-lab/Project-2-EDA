# DecodeLabs Internship – Project 2: Exploratory Data Analysis (EDA)

## Overview

This project was completed as part of the DecodeLabs Data Analytics Internship Program.

The objective was to perform Exploratory Data Analysis (EDA) on an e-commerce transaction dataset to identify trends, patterns, distributions, and outliers while generating meaningful business insights.

## Dataset Information

* Total Records: 1200
* Total Columns: 14
* Dataset Type: E-Commerce Transactions

## Tasks Performed

### Data Quality Analysis

* Checked missing values
* Verified duplicate records
* Reviewed dataset consistency

### Descriptive Statistics

* Calculated mean, median, minimum, and maximum values
* Analyzed Quantity, Unit Price, Total Price, and Items in Cart

### Trend Analysis

* Identified most ordered products
* Analyzed highest revenue-generating products
* Examined payment method preferences
* Reviewed order status distribution

### Outlier Detection

* Performed outlier analysis using the IQR method
* Identified unusual transaction values

## Key Findings

* Printer was the most frequently ordered product.
* Chair generated the highest revenue.
* Online payment was the most preferred payment method.
* Only a small number of outliers were detected.
* The dataset demonstrated strong overall quality with no duplicate records.
